## ③ 카카오 로그인

- [ ] https://developers.kakao.com → **내 애플리케이션 → 애플리케이션 추가하기**
  - 앱 이름 `물들임`, 사업자명 = 내 이름/닉네임
- [ ] **앱 설정 → 플랫폼 → Web 플랫폼 등록**
  - 사이트 도메인: `https://icebrain78.github.io`
- [ ] **카카오 로그인** 메뉴 → **활성화 ON**
  - [ ] **Redirect URI** 등록:
    ```
    https://ifjikwpmyfvzwetppnfx.supabase.co/auth/v1/callback
    ```
- [ ] **동의 항목**: 닉네임(profile_nickname) 등 필수만 설정
- [ ] **앱 키**에서 **REST API 키** 복사
- [ ] **카카오 로그인 → 보안 → Client Secret** 생성 → 복사 (활성화 상태 ON)

### B. Supabase
- [ ] Authentication → Providers → **Kakao** → Enable
- [ ] REST API 키 → **Client ID** 칸, Client Secret → **Client Secret** 칸 → **Save**

### C. 확인
- [ ] 웹에서 **카카오로 계속하기** → 카카오 로그인 → 갤러리 복귀 성공

## 막힐 때
- **redirect_uri_mismatch** → 리디렉션 URI 오타. 위 콜백 주소와 100% 같아야 함(끝 슬래시 포함 주의)
- **provider is not enabled** → Supabase Providers에서 Enable + Save 안 됨
- 로그인 후 흰 화면 → Supabase Redirect URLs에 사이트 주소 누락

각 단계에서 오류 화면 캡처해서 물어보면 바로 잡아드립니다.
